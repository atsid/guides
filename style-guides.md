## Overview 

> "-."

Style guides are built to show clients and team members the content visual direction of a product. This guide will include:

* Typography 
* Color
* Imagery 
* Icons
* UI Elements

## Why use this exercise? 

This exercise defines the "Look and Feel" for a project. It serves as a reference for both developers and designers to keep visual consistency throughout the product.

> "-."
